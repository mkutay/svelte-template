# svelte-template

A SvelteKit template that is generic enough to be used for any project, but also has some nice features like:

- TS
- biome
- vitest
- playwright
- tailwind
- shadcn-svelte
- neverthrow
- bun

## Add components

```bash
bunx shadcn-svelte@latest add button
```

## Supabase

NOTE: Supabase with version 2.67.1 is being used, as the latest self-hostable version is 2.67.1, to have parity in both environments.
